# Text Selection
1. Column Selection:
  `⌘` + `⇧` + `⬆`
  `⌘` + `⇧` + `⬇`
2. Mupliple selection
  * Quick add next:
    `⌘` + `D`
  * Undo Selection:
    `⌘` + `U`
  * Quick skip next:
    `⌘` + `K`
    `⌘` + `D`
